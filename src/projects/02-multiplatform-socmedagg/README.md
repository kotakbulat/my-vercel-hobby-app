# 📱 Unified Social Feed (Experimental)

A lightweight React experiment to simulate a **unified social media feed** by combining content from multiple platforms into a single scrolling experience.

---

## 🎯 Goal

This project is **not about production readiness**.

It exists to quickly validate one idea:

> “A unified feed from multiple platforms feels useful and engaging.”

---

## ⚠️ Constraints

To keep things fast, simple, and focused on UX validation:

- ❌ No authentication system  
- ❌ No database  
- ❌ No OAuth  
- ❌ No complex backend  
- ❌ No real scraping (avoid ToS issues)

### ✅ Allowed Data Sources

- Public APIs (optional)
- Mock data
- Static JSON
- Simple `fetch` calls

---

## 🏗️ Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- State management:
  - `useState`
  - Minimal React hooks
- Optional:
  - Simple mock API (if needed)

---

## 🧩 Core Idea

Simulate content from different platforms:

- 🎥 YouTube → videos  
- 📝 Twitter/X → text posts  
- 🖼 Instagram → images / reels  

Then merge everything into **one unified feed**.

---

## 📦 Data Strategy (Important)

Instead of real integrations, use mock data.


---

## 🧪 Features
- Mixed feed from multiple platforms
- Infinite scroll or “Load More”
- Platform labels/icons
- Randomized content order
